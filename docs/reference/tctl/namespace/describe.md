The `tctl namespace describe` command describes a Namespace.

`tctl namespace describe --namespace-id <id>`

The following modifier controls the behavior of the command.

### `--namespace-id`

_Required modifier_

How to specify the ID of a Namespace.

**Example**

```
tctl namespace describe --namespace-id <id>
```
